# Landing Page SERCOTEC Santiago

Este proyecto corresponde a la entrega de la pauta universitaria para la creación de una landing page profesional, modular y responsiva para SERCOTEC Santiago, desarrollada 100% con HTML5, CSS3 y JavaScript nativo, sin frameworks completos.

## Objetivo
Desarrollar un sitio web moderno, visualmente atractivo y funcional, que represente a SERCOTEC Santiago y sus servicios, siguiendo las mejores prácticas de desarrollo web y cumpliendo con todos los requisitos de la pauta.

## Estructura y modularidad
- **HTML5**: Estructura semántica, ordenada y validada.
- **CSS3**: Modularizado por sección (hero, sobre nosotros, servicios, testimonios, FAQ, contacto, footer), uso de variables CSS y diseño responsivo desde el inicio.
- **JavaScript nativo**: Lógica separada por sección, consumo dinámico de datos desde archivos JSON, validación de formularios y componentes interactivos (carrusel, acordeón FAQ).
- **Carpetas**:
  - `assets/css/`: Estilos por sección.
  - `assets/js/`: Scripts por sección.
  - `assets/imagenes/`: Imágenes institucionales y de usuarios.
  - `data/`: Archivos JSON para servicios, testimonios y preguntas frecuentes.

## Componentes y funcionalidades
- **Navbar responsive**
- **Hero institucional**
- **Sección Sobre Nosotros** con carrusel misión/visión/valores
- **Servicios**: Tarjetas modernas, datos dinámicos desde JSON
- **Testimonios**: Carrusel, imágenes randomuser.me, datos desde JSON
- **Preguntas frecuentes**: Acordeón dinámico, datos desde JSON
- **Formulario de contacto**: Validación en tiempo real, campos requeridos y feedback visual
- **Footer**: Moderno, con logo y enlaces a redes

## Cumplimiento de la pauta
- Todo el contenido y los componentes están en español.
- No se usaron frameworks completos, solo JS nativo y librerías ligeras para componentes puntuales.
- Consumo de datos dinámico desde archivos JSON.
- Modularidad total en carpetas y archivos.
- Diseño 100% responsivo y visualmente coherente.
- Imágenes de usuarios en testimonios desde randomuser.me.
- Validación visual y estructural tras cada cambio.
